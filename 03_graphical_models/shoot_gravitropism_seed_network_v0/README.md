# Shoot Gravitropism Seed Network v0

This folder contains a hypothesis-level seed network, not curated biology.

It is a teaching and planning scaffold for thinking about shoot gravitropism
over development and evolutionary time. It should not be treated as a curated
FLASH-P network, and no edge should be cited as biological evidence until a
specific reference has been added and checked.

## Purpose

The goal is to sketch the first causal shape of a shoot gravitropism model:

1. a physical stimulus is applied,
2. shoot gravity-sensing tissues perceive it,
3. signalling modules redistribute auxin,
4. auxin asymmetry changes cell expansion,
5. curvature changes through time,
6. genetic and family-level differences may alter the response.

## Files

- `candidate_nodes.csv`: Proposed nodes grouped by layer.
- `candidate_edges.csv`: Hypothesis-level signed edges.
- `evidence_gaps.md`: What must be supported by literature before curation.

## Sign Convention

- `1` means activation, promotion, or positive causal influence.
- `-1` means inhibition, repression, or negative causal influence.

Signed edges here are causal summaries for hypothesis building. They are not
proof of direct molecular mechanisms.

## Status

Every edge is currently marked `hypothesis`, and every edge has
`reference_needed` set to `yes`.

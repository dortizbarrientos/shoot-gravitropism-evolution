# Graphical Models

Use this folder for pathway sketches, causal graph drafts, standards-oriented
SBGN files, and generated visual previews.

Current conventions:

- Treat `candidate_nodes.csv` and `candidate_edges.csv` as the editable source
  tables for each hypothesis network.
- Treat SBGN files and visual exports as derived artifacts.
- Keep generated previews close to the model version they describe, usually in
  an `exports/` folder.
- Use `exports/working_previews/` for inspection figures and
  `exports/publication_drafts/` for selected figure elements worth refining.
- Label every network as hypothesis-level, review-supported,
  primary-paper-supported, or curated.

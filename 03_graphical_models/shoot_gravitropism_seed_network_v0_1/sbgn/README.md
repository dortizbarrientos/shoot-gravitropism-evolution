# SBGN Drafts For Shoot Gravitropism Seed Network v0.1

This folder contains SBGN Activity Flow draft files for the v0.1 shoot gravitropism seed network.

## Current Draft

- `shoot_gravitropism_seed_network_v0_1_causal_spine_af.sbgn`

This is an SBGN Activity Flow draft.

It represents only the causal spine edges where `include_in_first_sbgn = yes` in `candidate_edges.csv`.

It does not represent genotype modulation edges, context edges, or classification edges.

## Scientific Status

This draft is hypothesis-level, not curated biology.

No edge should be treated as curated until a specific reference is added.

Signed edges are causal summaries for modelling. They are not proof of a direct molecular mechanism.

## Validation Needed

This file should be opened in Newt for visual validation.

It should not be used as PSoup input until it has been visually checked and validated as a proper SBGN Activity Flow file.

# FLASH-P Exploration Notes

This folder is organized as a learning path for students reading the repository
without running the full pipeline.

## Suggested Reading Order

1. `00-overview/`
   - Start here for a broad repository orientation.

2. `01-pipeline/`
   - Read this next to understand the Plant pipeline stages and handoff files.

3. `02-data-structures/`
   - Read this before editing or creating JSON. It explains edges, nodes,
     perturbation tests, `network.json`, and RWR inputs.

4. `03-learning-examples/`
   - Read this while inspecting the toy files under
     `07_flashp_learning/learning_examples/minimal_light_network/`.

## Safety Principle

These notes are for inspection and learning. They do not require running the
full FLASH-P pipeline.
